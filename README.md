Installation

Clone the repository:

git clone https://github.com/your-username/quiz-app.git
cd quiz-app

Install dependencies:

npm install

Run the project

Start the development server:

npm run dev

Application will run at:

http://localhost:5173

E2E Testing

Project uses Playwright for end-to-end testing.

Run tests:

npx playwright test

Run tests with UI:

npx playwright test --ui

Run a specific test:

npx playwright test answerQuestion.spec.ts
